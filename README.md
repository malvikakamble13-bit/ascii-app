# 3D ASCII Text Animation App

An interactive 3D ASCII text animation web application built with **React**, **Three.js**, and **Vite**, deployed on **Vercel**. 

Users can customize text, colors, and font sizes in real time with interactive controls overlaying a 3D animated ASCII canvas.

## 🌐 Live Demo

Check out the live interactive application here:
👉 **[Live Demo](https://ascii-app-beige.vercel.app)**

---

## ✨ Features

* **3D ASCII Rendering**: Converts 3D scene elements into customizable ASCII character art in real time.
* **Interactive UI Controls**:
  * 🔤 **Custom Text**: Type any word or phrase to see it update instantly on the 3D canvas.
  * 🎨 **Color Picker**: Change text and particle accent colors dynamically.
  * 📏 **Font Size Slider**: Adjust character sizing on the fly.
* **High Performance**: Built with Vite and Three.js for smooth animations and high frame rates.

---

## Tech Stack

* **Frontend Framework:** React 19
* **3D Engine:** Three.js
* **Build Tool:** Vite
* **Deployment Platform:** Vercel
